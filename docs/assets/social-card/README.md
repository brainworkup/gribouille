# Social card

`../images/social-card.png` is the 1200×630 Open Graph / Twitter card image
referenced site-wide from `docs/_quarto.yml`.
It is a committed artefact, not regenerated by `quarto render`.

To rebuild it:

```sh
./build.sh
```

`build.sh` first renders the homepage penguins figure (`examples/gribouille.typ`,
wrapped with `docs/_typst-preamble.typ` and the light-scheme colours from
`docs/_brand.yml`) to `penguins-light.svg`, then screenshots `social-card.html`
with headless Chromium (Playwright).
It needs `quarto` (for the bundled Typst) and `npx`; Chromium is downloaded on
first run.

Inputs that affect the result: `examples/gribouille.typ`, `docs/_typst-preamble.typ`,
`docs/_brand.yml` (light palette), `docs/assets/fonts/*.woff2`, `docs/assets/images/logo.svg`.
After changing any of those, re-run `./build.sh` and commit the new PNG.
